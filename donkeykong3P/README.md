# Donkey Kong Arcade Clone

## Description

This project is a recreation inspired by the classic **Donkey Kong Arcade** game, developed in **Godot Engine**.
The game includes multiple levels, arcade-style gameplay, enemies, hammer mechanics, score system, lives system, retro audio, and progressive difficulty.

The objective of the game is to guide the player through platforms and ladders while avoiding barrels and enemies in order to rescue the princess.

---

## Features

* 3 playable levels
* Progressive difficulty system
* Barrel spawning system
* Random barrel ladder drops
* Hammer power-up
* Enemy fireballs
* Vulnerable enemies while hammer is active
* Score system
* Lives system
* Pause menu
* Arcade sound effects and music
* Level transition screens
* Princess heart transition
* Game Over and Win systems
* Arcade Animations

## Gameplay Systems

### Barrel System

Barrels are spawned continuously by Donkey Kong and move through platforms. Some barrels can randomly descend ladders to increase difficulty.

### Hammer System

The player can collect hammers to temporarily destroy barrels and enemies.

### Enemy System

Fireball enemies patrol platforms and damage the player on contact. While the hammer is active, enemies become vulnerable.

### Progression

The game contains multiple levels with increasing complexity, more dangerous layouts, and additional hazards.
